Face Finder is system for face recognition written in C++. It uses OpenCV library to get image from web camera.<br>
Image is transformed to Eigenfaces space and learn in Counter-Propagation Neural Network. <br>
This documentation describes all classes need to build working system.<br>
The example of system implemetation is program face-finder with GUI written in QT4.5 and was not documented here.
